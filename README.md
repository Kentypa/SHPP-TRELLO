# Installiation

## Clone repository

```bash
git clone https://github.com/Kentypa/SHPP-TRELLO
```

## Open folder

```bash
code ./SHPP-TRELLO
```

## Install dependencies

```bash
yarn install
```

## Make .env file

```bash
cp .env.example .env
```

## Write into .env API key

## Run project

```bash
yarn dev
```
